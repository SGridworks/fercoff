# Changelog

All notable changes to FERCoff will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.0] - 2026-03-03

### Added
- Initial open-source release
- Complete v3.1 white paper with performance benchmarks
- OpenClaw security hardening guidance (ClawJacked vulnerability)
- Tailscale vs. WireGuard vs. data diode progression
- Output sanitization and hallucination warnings
- NERC CIP audit reality check section
- Labor cost estimates (not just hardware)
- USB scaling reality check

### Changed
- Repository name: FERCoff (was "Secure AI Sandbox")
- License: Dual MIT (code) + CC-BY-SA 4.0 (docs)

### Fixed
- All factual errors from v2.0 review
- Qwen model sizes (3B/7B, not 4B/9B)
- FileVault encryption spec (XTS-AES-128)
- Power consumption figures (~40-65W, not ~20W)
- Statutory citations (FAST Act 2015, not EPAct 2005)

## [3.0.0] - 2026-03-03

- Complete restructure from compliance white paper to practical guide
